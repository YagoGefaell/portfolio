# Yago Gefaell – Portfolio

Personal portfolio website built with [Astro](https://astro.build).

## Features

- **Hero / About** — Introduction with name, title, and bio
- **Projects** — Project cards with links to repositories or live demos
- **Skills** — Technology icons (HTML, CSS, JS, TypeScript, React, Vue, Java, Spring, Docker, Git, Node.js, SQL)
- **Contact** — Links to GitHub, email, and LinkedIn
- Responsive design with a dark theme

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Tech Stack

- [Astro](https://astro.build) — Static site framework
- TypeScript
- CSS (scoped component styles + global variables)
